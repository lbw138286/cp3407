# US4: Edit and Delete Expenses

Status: Done  
Iteration Completed: Iteration 1  
Project: Smart Expense Tracker Web Application

## Summary

US4 was completed to allow users to manage existing expense records. Users can edit incorrect or outdated expense information and delete records that are no longer needed. After each edit or delete action, the expense table is refreshed and local storage is updated.

## Completed Functionality

- Edit button added to each expense row.
- Delete button added to each expense row.
- Selected expense data can be loaded back into the form for editing.
- Existing expense data can be updated.
- Expense records can be deleted.
- Local storage is updated after edit and delete operations.

## Value to the Project

This user story improves the reliability of the expense tracker because users can correct mistakes and remove unnecessary records. It makes the application more practical for daily expense management.
