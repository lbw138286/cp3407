# Practical 6: GitHub Tracking Plan

Project: Smart Expense Tracker Web Application  
Iteration: Iteration 2

## GitHub Project Board Columns

| Column | Purpose |
|---|---|
| Todo | Tasks planned but not started yet. |
| In Progress | Tasks currently being worked on. |
| Done | Tasks fully completed and checked. |

## Labels to Use

| Label | Meaning |
|---|---|
| iteration-2 | The issue belongs to Iteration 2. |
| todo | The task has not started. |
| in-progress | The task is currently being developed or reviewed. |
| done | The task is completed. |
| frontend | UI-related work. |
| data-storage | Local storage or persistence work. |
| testing | Testing and verification work. |
| documentation | Documentation and GitHub Pages updates. |
