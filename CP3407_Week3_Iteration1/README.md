## Completed User Story

### US3 - Add Expenses

**User Story:** As a user, I want to add daily expenses so that I can track my spending habits.

**Implemented functionality:**
- User can enter an expense title.
- User can enter an amount.
- User can select a category.
- User can select a date.
- Expense records are displayed in a table.
- Expense records are saved in browser localStorage.
- Basic validation prevents empty title, invalid amount, missing category, or missing date.

## How to Run

1. Open the `src/index.html` file in a web browser.
2. Fill in the expense form.
3. Click **Add Expense**.
4. Confirm that the new expense appears in the expense table.
