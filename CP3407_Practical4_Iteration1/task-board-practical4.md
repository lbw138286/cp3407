## Label Rules

- **todo**: task has been planned but not started.
- **in-progress**: task is currently being implemented or reviewed.
- **done**: task has been completed and checked.
- **feature**: implementation task related to user functionality.
- **design**: task related to diagrams or system structure.
- **testing**: task related to validation and checking.
- **github**: task related to issues, pull requests, commits, or tracking.
