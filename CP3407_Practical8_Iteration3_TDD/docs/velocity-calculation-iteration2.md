# Iteration 2 Actual Velocity Calculation

## Team capacity

The team has three members. Each member has one working day per week across four working days.

```text
Team capacity = 3 team members × 4 working days
Team capacity = 12 person-days
```

## Completed user stories in Iteration 2

| User Story | Estimate | Status |
|---|---:|---|
| US5: Categorize Expenses | 6 person-days | Done |
| US6: Set Monthly Budgets | 6 person-days | Done |

## Actual velocity

Velocity is calculated based on the completed user stories only.

```text
Actual velocity = completed US5 + completed US6
Actual velocity = 6 + 6
Actual velocity = 12 person-days
```

## Result

The actual velocity of Iteration 2 is **12 person-days**.

This value is used to update the backlog and plan the workload for Iteration 3.
